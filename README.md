# 🚀 ToolNova - Access 50+ Free AI Tools

ToolNova is a premium, free-forever AI tools hub designed for students, professionals, and creators. We provide 50+ AI-powered utilities for PDF manipulation, writing assistance, image editing, and study aids.

**URL:** [https://www.toolnovahub.com](https://www.toolnovahub.com)

## ✨ Available AI Tools

ToolNova offers a wide range of free AI tools, including:

### 📝 Writing & Content
- **Text Summarizer**: Summarize articles and documents instantly.
- **Paraphraser**: Rewrite text while maintaining meaning.
- **Grammar Fix**: specialized grammar and spell checker.
- **Essay Writer**: Generate structured essays on any topic.
- **Email Writer**: Create professional business emails.

### 📚 Study & Education
- **Homework Solver**: Step-by-step solutions for math and science.
- **Flashcard Maker**: Create study flashcards from notes.
- **Quiz Generator**: Generate quizzes from any text.
- **Concept Explainer**: Simplify complex topics.

### 📄 PDF Tools
- **Merge PDF**: Combine multiple PDFs into one.
- **Split PDF**: Extract pages from PDF files.
- **Image to PDF**: Convert JPG/PNG to PDF documents.

### 🖼️ Image Tools
- **Image Compressor**: Reduce file size without quality loss.
- **Background Remover**: (Coming soon)
- **Image Converter**: JPG to PNG, PNG to JPG.

## 🤖 AI Discovery & API

ToolNova is optimized for AI agents (ChatGPT, Gemini, Perplexity).

- **AI Plugin Manifest**: `/.well-known/ai-plugin.json`
- **OpenAPI Spec**: `/openapi.yaml`
- **Tools Discovery API**: `/api/tools` (Returns list of all available tools)
