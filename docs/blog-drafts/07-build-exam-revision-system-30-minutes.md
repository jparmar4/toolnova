## 30-minute setup plan

### 0–10 min: Prioritize topics
- List all chapters
- Mark each as: High / Medium / Low confidence
- Allocate more sessions to weak + high-weight chapters

### 10–20 min: Build schedule
- Create weekly slots by subject
- Keep 60–90 minute focused blocks
- Add 10-minute breaks and one weekly buffer block

Use: https://www.toolnovahub.com/tools/timetable-generator

### 20–30 min: Add active recall layer
- Convert key concepts into flashcards
- Create a quick quiz set per chapter

Use:
- https://www.toolnovahub.com/tools/flashcard-maker
- https://www.toolnovahub.com/tools/quiz-generator

## Daily revision template

1. 15 min recap (yesterday topics)  
2. 45 min deep study (new concept)  
3. 20 min active recall (flashcards/quiz)  
4. 10 min error log and next-day plan

---

## Common planning mistakes

- Overloading one day with too many subjects
- No buffer for backlog topics
- Passive reading without recall practice

---

## FAQ

### How many hours should I revise daily?
Consistency matters more than extremes. Start with 2–4 focused hours.

### Should I revise all subjects every day?
No. Rotate based on priority and exam timeline.

### How do I track progress?
Use completion percentage + quiz score trends.

---

## Final takeaway

A simple structured routine beats random long study sessions. Plan once, then run the system daily with active recall.
