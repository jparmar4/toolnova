# Publish Day SOP (10–15 Minutes per Post)

Use this SOP every time a blog post is published.

---

## A) Pre-publish (5 minutes)
1. Copy final draft into CMS editor.
2. Set slug from `SEO-META-PACK-12-POSTS.md`.
3. Paste meta title + meta description.
4. Add:
   - Author line
   - Last updated date
   - Featured/OG image
5. Insert internal links:
   - 1 primary tool page
   - 2 secondary tool/category pages
   - 1 related blog post
6. Add 3 FAQs at the end.
7. Ensure CTA appears near top and near conclusion.

Quick pass checks:
- H1 matches intent
- No broken links
- Mobile readability okay

---

## B) Publish (1 minute)
1. Click Publish.
2. Open live URL in incognito.
3. Confirm status 200 and page content loads fully.

---

## C) Immediate indexing push (3–4 minutes)
1. Google Search Console:
   - URL Inspection → Request Indexing
2. Bing Webmaster:
   - Submit URL
3. Add published URL to weekly tracker notes:
   - `docs/SEO-WEEKLY-KPI-TRACKER.csv`

---

## D) 7-day optimization loop (2–4 minutes)
1. Check impressions/clicks/CTR for the new post.
2. If impressions high but CTR low:
   - Rewrite title (clearer benefit + keyword)
   - Tighten meta description (outcome + specificity)
3. Add 1 fresh internal link from latest post to an older money page.

---

## E) Stoplight decision (fast)
- **Green:** indexed + impressions rising → continue schedule
- **Yellow:** indexed but low CTR → adjust title/meta
- **Red:** not indexed after 7 days → re-submit + verify crawl/sitemap/canonical
