# 1-Click Publishing Checklist (12 SEO Posts)

Use this checklist for every post before/after clicking Publish.

---

## Universal Pre-Publish Checklist (copy for each post)
- [ ] Correct slug added
- [ ] Meta title added
- [ ] Meta description added
- [ ] H1 matches search intent
- [ ] Intro contains quick answer block (within first 120 words)
- [ ] 4â€“6 internal links added
  - [ ] At least 1 link to primary target tool URL
  - [ ] At least 1 link to category page
  - [ ] At least 1 link to related blog post
- [ ] 3 FAQs added at end
- [ ] Author + Last Updated date visible
- [ ] CTA button/link near top and near conclusion
- [ ] Featured image/OG image assigned

## Post-Publish Checklist (immediately after publish)
- [ ] Open live URL and confirm page loads (200)
- [ ] Confirm canonical URL is correct
- [ ] Confirm OG/Twitter tags render
- [ ] Submit URL in Google Search Console (URL Inspection)
- [ ] Submit URL in Bing Webmaster URL submission
- [ ] Add URL to weekly KPI tracker notes
