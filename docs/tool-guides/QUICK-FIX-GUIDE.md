# Formula Generator - Quick Fix Guide

## 🚨 Common Issues & Instant Solutions

### Problem: Getting theory/headings instead of actual formulas

**✅ QUICK FIX:**
1. Switch to **"📊 Table Format"** in the Output Format dropdown
2. Add this to your input: "Show actual mathematical equations, not descriptions"
3. Be specific: "Trigonometric identities" not "Trigonometry"

**Example Input:**
```
Trigonometric sum and difference identities
(Include: sin(A±B), cos(A±B), tan(A±B))
```

---

### Problem: Table format not working / no pipe characters

**✅ QUICK FIX:**
In your input, explicitly state:
```
Present as a Markdown table with columns: Formula Name | Equation | Notes
```

**Already fixed in latest version** - just select Table Format.

---

### Problem: Missing important formulas

**✅ QUICK FIX:**
List exactly what you need:
```
Chemical Equilibrium formulas including:
- Kc and Kp expressions
- Relationship between Kp and Kc
- Reaction quotient Q
- Gibbs free energy relations
- van't Hoff equation
- Solubility product Ksp
```

---

### Problem: Too basic / too advanced

**✅ QUICK FIX:**
Specify the level:
- "High school algebra" or "Undergraduate real analysis"
- "NEET/JEE level" or "AP Chemistry"
- "Basic calculus" or "Advanced differential equations"

---

### Problem: Output too short

**✅ QUICK FIX:**
Use these phrases:
- "Generate a **complete** formula sheet..."
- "Include **all standard** formulas for..."
- "Comprehensive list of..."

---

### Problem: Unclear notation / hard to read

**✅ QUICK FIX:**
Add to your input:
```
Use clear notation with:
- Subscripts for indices
- Superscripts for exponents
- Greek letters for standard symbols
- Standard [subject] notation
```

---

### Problem: No examples when toggle is enabled

**✅ QUICK FIX:**
Explicitly request:
```
Include a brief numerical example for each formula
```

## ✅ Quality Checklist

Your output should have:
- [ ] Actual equations (e.g., `Kc = [C]^c[D]^d / [A]^a[B]^b`)
- [ ] At least 10-15 formulas for comprehensive topics
- [ ] Proper mathematical notation
- [ ] Brief usage notes (table/explained format)
- [ ] No "WORKING" or "FINAL ANSWER" headers

If any are missing → copy your input and select Table Format
